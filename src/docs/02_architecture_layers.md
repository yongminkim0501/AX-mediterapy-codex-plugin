# Architecture Layers

1. Evidence Base Layer stores public source references, lifecycle, scope, and baseline soft insight symbols.
2. Plan Structuring Layer validates and normalizes a JSON plan against the canonical contract.
3. Ontology-Guided Symbol Formation Layer emits atomic symbols first and derives composite symbols from components only when needed.
4. Constraint & Conflict Projection evaluates hard constraints with Z3 and soft conflicts with symbol set diffs.
5. Report Layer converts deterministic results into marketer-facing status and questions.
